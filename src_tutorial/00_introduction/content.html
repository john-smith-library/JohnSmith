John Smith is a lightweight no-magic frontend library for building interactive UI.

Core Principles:

  • TypeScript-based, strongly typed wherever possible.
  • Pure separation of concerns via classic View-ViewModel pattern.
  • View and ViewModel nesting to support hierarchical application structure.
  • Change propagation from ViewModel directly to DOM via Observable channels.
  • JSX syntax for view declaration.

These principles lead to the following features:

  • Explicit DOM manipulation, no extra rendering cycles, no Shadow DOM.
  • Explicit reactive model, no hidden change detection, no digest/ingest cycles.
  • Views composing in the Build time, no runtime template parsing.
